Relógio Digital

Instruções de uso

Primeiramente, vire o relógio coloque na opção do maio (Sem som).

No menu principal, você verá três opções: basic, bonus e delay.

basic, ou seja, a opção sem acréscimo

Vemos aqui que está configurado para um jogo de 5 minutos contra 5 minutos. Caso você deseje alterar essa configuração, clique uma vez no botão reset e navegue entre horas, minutos e segundos usando o botão start.

Configurou o tempo como deseja? Aperte só uma vez o botão confirm.

Para restaurar o tempo segure o botão reset durante 3 segundos. Ele vai voltar ao menu principal. Para selecionar o modo básico novamente, aperte o botão confirm.

bonus, ou seja, a opção com acréscimo

Aqui mostra 3 minutos + 2 segundos. Caso deseje mudar é a mesmo procedimento. Aperte uma vez no botão confirm e navegue entre horas, minutos e segundos com o botão start. Altere o acréscimo caso deseje e aperte de novo o botão confirm para confirmar. Para o tempo começar a correr, aperte o botão start. Para restaurar, segure o botão reset e pronto voltamos pro menu principal.
